# CareConnect - Mini Healthcare Support Web App

Care4All is a simple healthcare support prototype designed for NGOs to help patients connect with volunteers.

Features
- Patient support request form
- Volunteer registration form
- AI FAQ chatbot concept
- Automated response system
- Request summary counter

Tech Stack
HTML
CSS
JavaScript

AI / Automation Idea
The application includes a rule-based chatbot that answers basic healthcare questions automatically. In a real NGO system, this could be integrated with AI models to provide intelligent health guidance and triage patient requests.

Use Case
NGOs can use this platform to collect patient support requests, manage volunteer registrations, and provide instant automated assistance to users.
